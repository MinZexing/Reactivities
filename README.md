# Reactivities

Reactivities is a full stack web app (.NET & React).  
It allows users to browse activities/events and manage their account by logging in and out.

> ⚠️ This project is still in progress. The repository is public so you can follow along as new features are added.

---

## Features (in progress)

- View a list of activities with basic info
- View details for a single activity (WIP)
- User authentication: login / logout
- Responsive UI with modern React patterns
- Backend API for activities and users (C# .NET ASP.NET Core)

---

## Tech Stack

**Frontend**

- React
- TypeScript

**Backend**

- C#
- ASP.NET Core Web API

---

## Getting Started

> Note: Setup steps may change as the project evolves.

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd API
dotnet restore
dotnet watch
```
